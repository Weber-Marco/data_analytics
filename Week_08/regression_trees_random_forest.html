Differences Observed:¶

The tree with max_depth=5 will have more levels and splits compared to the tree with max_depth=3. A deeper tree can model more complex relationships in the data, which might improve the fit on the training data.

The visual representation of the tree with higher max depth will show more nodes and branches. This can result in a lower R-squared value on the test set, indicating poorer generalization to new data. (Overfitting)
